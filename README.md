# InstaUnfollow - Quem não te segue de volta?

Uma extensão simples e poderosa para o Chrome que identifica quem você segue no Instagram mas que não te segue de volta.


## 🚀 Funcionalidades

- **Scan em Tempo Real**: Veja os resultados aparecendo enquanto a extensão processa sua lista.
- **Seguro**: Implementa limites de taxa (rate limiting) e pausas inteligentes para evitar bloqueios da conta.
- **Fácil de Usar**: Interface limpa com modo escuro e design premium.
- **Unfollow em Massa**: Opção para remover todos os não-seguidores automaticamente com intervalos de segurança.

## 🛠️ Instalação (Desenvolvedor)

1. Clone este repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/instagram-unfollower.git
   ```
2. Abra o Chrome e vá para `chrome://extensions/`.
3. Ative o **Modo do desenvolvedor**.
4. Clique em **Carregar sem compactação** e selecione a pasta do projeto.
